# 🚀 MailPilot AI

MailPilot AI is an AI-powered email automation platform that helps users generate personalized emails, manage contacts, and automate email campaigns.

## ✨ Features

- 🔐 User Authentication (Signup & Login)
- 🔑 JWT Authentication
- ☁️ MongoDB Atlas Integration
- ⚛️ React Frontend
- 🚀 Express.js Backend
- 📧 AI Email Automation (Coming Soon)
- 📂 CSV Contact Upload (Coming Soon)
- 📊 Campaign Dashboard (Coming Soon)

## 🛠️ Tech Stack

### Frontend

- React
- Vite
- Tailwind CSS
- React Router
- Axios
- React Hook Form

### Backend

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
- bcryptjs

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Pavanvanjarapu12/MailPilot-AI.git
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run dev
```
